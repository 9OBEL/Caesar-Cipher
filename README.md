# Caesar Cipher Python Implementation

A simple, efficient, and lightweight Python script designed to perform Caesar cipher encryption and decryption. This implementation handles both uppercase and lowercase letters while keeping special characters and spaces unchanged.

## Features

*   **Customizable Shift:** Supports any integer shift value between 1 and 25.
*   **Case Sensitivity:** Automatically preserves the original letter casing.
*   **Non-Alphabetic Support:** Leaves spaces, numbers, and punctuation intact.
*   **Simple API:** Includes easy-to-use functions for both encryption and decryption.
